Welcome to EEGraSP's documentation!
===================================

**EEGraSP** is a Python library to analyze EEG data based on Graph Signal Processing (GSP) tools. The repository can be found at <https://gitlab.com/gsp8332409/eegrasp>. 

.. note::
   This project is under developement. Contributions through merge requests are welcomed.
